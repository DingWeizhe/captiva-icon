# 使用教學
1. 安裝 [liteicon](https://freemacsoft.net/liteicon/)
2. 重開 Mac ， 並在開機時按下 cmd + r 進入修復畫面
3. 在 Utilities > Terminal 開啟文字命令視窗
4. 輸入 csrutil disable 關閉 System Integrity Protection 保護機制
5. 開啟 liteicon 無痛更換 icon
